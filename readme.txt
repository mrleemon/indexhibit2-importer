== Description ==

Import categories, users, posts, comments, and links from an Indexhibit 2 site.

== Installation ==

1. Upload the `indexhibit-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on Indexhibit
